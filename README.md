Разница между launch и - async — для параллельных вычислений с результатом.
launch — для фоновых задач без возврата значения.
Suspend-функции в Kotlin — это специальные функции, предназначенные для работы с корутинами. Они позволяют приостанавливать выполнение кода без блокировки потока, что важно для асинхронных операций
Диспетчер - нужны для разделения потоков выполнения по типам задач
Если не обработать исключение в корутине (Coroutine) в Kotlin, это может привести к сбою приложения.
Автоматическая отмена корутин в Kotlin происходит через механизм, связанный с областями видимости.
